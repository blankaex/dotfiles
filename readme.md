## Requirements

* [pyinfra](https://github.com/Fizzadar/pyinfra)

## Installation

```
$ pyinfra @local deploys/izanagi.py
```
Or alternatively over SSH:
```
$ pyinfra [HOST] deploys/izanagi.py --user [SSH USERNAME] --key [PATH] [--key-password [PASSWORD]]
```
